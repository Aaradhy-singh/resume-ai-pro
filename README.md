ResumeAI Pro — Resume Analysis Intelligence
A browser-based resume diagnostic tool that runs 8 analysis engines to identify skill gaps, ATS compatibility issues, and missing impact metrics. No account required. Your resume never leaves your browser.

Features - 
* Skill gap analysis against 300+ roles across 38 industries
* ATS compatibility scoring with keyword alignment
* Impact scoring for resume bullet points
* Career stage detection
* GitHub portfolio analysis
* Prioritised action plan with effort estimates
* PDF export for results and action plan
* Career Explorer — compare your profile against target roles

Tech Stack - 
* React + TypeScript + Vite
* TailwindCSS
* jsPDF
* Octokit (GitHub API)
* Custom rule-based analysis engines
